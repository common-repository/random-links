== Description ==

Create one link that rotates multiple links.  Cloak single affiliate links and setup short links that can redirect to one link or many.  Redirect multiple links in random or sequential order and view statistics on all clicks.


== Installation ==

1. Upload and Activate Plugin
2. Goto Settings > Random Links
3. Create a link name for admin purposes
4. Enter 1 to unlimited urls to rotate
5. Choose display method: Random or sequential
6. Save
7. Copy link to use any way you wish.
